# Privacy Policy for SimpleShow for xLights

SimpleShow is built with a strict "privacy-by-design" philosophy. Because this application runs entirely in your local web browser, your data never leaves your machine.

### 1. No Data Collection
We do not collect, store, track, or share any personal information, usage metrics, or analytics. There are no "phone home" features, tracking pixels, or telemetry scripts embedded in this software.

### 2. Local File Processing
All file parsing (such as `xlights_rgbeffects.xml` or `.xsq` sequence files), audio analysis, and shader rendering happen entirely client-side within your browser. No files or media are ever uploaded to an external server.

### 3. Third-Party Hosting (GitHub)
While SimpleShow does not collect data, the project is hosted via GitHub. GitHub may collect basic server logs and technical data standard for web hosting. You can review their terms in the [GitHub Privacy Statement](https://github.com).

### 4. Changes to This Policy
If future updates require network features (such as an optional update checker), this policy will be updated explicitly in advance, and features will operate on a strict opt-in basis.

**Contact:** If you have questions about this policy, please open an Issue on this repository.
